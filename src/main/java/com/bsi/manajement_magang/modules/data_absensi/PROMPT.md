# Daftar Fitur Modul Data Absensi

## Mentor
- Filter absensi berdasarkan status (Hadir, Sakit, Izin, Alfa, Semua).
- Filter absensi berdasarkan pencarian nama mahasiswa.
- Mengekspor rekap absensi ke format Excel/CSV.
- Memverifikasi absensi mahasiswa (Setujui atau Tolak/Hapus).
- Membaca daftar (list) absensi mahasiswa.
- Melihat file lampiran (surat keterangan) yang diunggah oleh mahasiswa.
- Melihat statistik absensi keseluruhan dengan filter (Total Hadir, Total Izin/Sakit).

## Mahasiswa
- Melakukan absensi harian:
  - Hadir.
  - Sakit (Wajib unggah file PDF/Image maks 10MB beserta form keterangan).
  - Izin (Wajib unggah file PDF/Image maks 10MB beserta form keterangan).
- Melihat status Alfa (tanpa keterangan) secara otomatis.
- Melihat riwayat absensi harian terakhir.
- Melihat statistik absensi pribadi (Jumlah Hadir, Izin, Sakit, Alfa).
