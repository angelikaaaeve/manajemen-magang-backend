# Fitur data_mahasiswa

**POST** `/api/mahasiswa`
Definisi: Endpoint untuk addStudent

**PUT** `/api/mahasiswa/edit-by-mentor/{id}`
Definisi: Endpoint untuk editStudent

**GET** `/api/mahasiswa`
Definisi: Endpoint untuk listStudents

**GET** `/api/mahasiswa/statistik`
Definisi: Endpoint untuk getStudentStatistics

**GET** `/api/mahasiswa/{id}`
Definisi: Endpoint untuk getStudentDetail

**GET** `/api/mahasiswa/sisa-waktu-magang`
Definisi: Endpoint untuk getSisaWaktuMagang
