# Daftar Fitur Modul IAM (Identity and Access Management)

## Autentikasi & Otorisasi
- Proses Registrasi (Pendaftaran pengguna baru).
- Proses Login (Autentikasi dengan Email/Password).
- Pengecekan status sesi/Token dan identitas Role (Mahasiswa / Mentor).
- Pembaruan (Update) profil pengguna dan kata sandi.
