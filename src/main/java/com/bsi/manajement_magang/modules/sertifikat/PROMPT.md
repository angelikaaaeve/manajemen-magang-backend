# Daftar Fitur Modul Sertifikat

## Mentor
- Filter sertifikat berdasarkan status (Sudah Diunggah, Belum Diunggah, Semua Status).
- Filter sertifikat berdasarkan pencarian nama mahasiswa.
- Mengunggah file sertifikat kelulusan magang untuk mahasiswa.
- Update status otomatis menjadi "Sudah Diunggah" jika file berhasil disimpan, atau "Belum Diunggah" jika kosong.
- Melihat statistik sertifikat dengan filter:
  - Total Sertifikat Diunggah.
  - Total Sertifikat Belum Diunggah.
  - Total Keseluruhan Sertifikat.

## Mahasiswa
- Melihat atau mereview sertifikat yang sudah diterbitkan.
- Mengunduh (download) file sertifikat yang diunggah mentor.
- Melihat detail spesifikasi sertifikat (Tanggal Terbit, Format Berkas).
